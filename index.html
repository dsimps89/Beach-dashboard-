Live sources used

  • Open-Meteo Forecast API: air temp, rain, wind, gusts, UV, cloud cover.
  • Open-Meteo Marine API: wave height, wave period, wave direction where available.
  • Open-Meteo Archive API: optional 10-year same-date heat trend.
  • Weekend/season priors are used as Bayesian-style crowd evidence.
